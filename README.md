# Blog
- MVC basit olarak geliştirilmiş bir blog.

# Özellikler
- Giriş/Kayıt bölümleri bulunmaktadır.
- Ana sayfada slider bölümü, blog yazıları, kategoriler ve arşiv (sidebar) bölümleri yer alır.
- Yönetim paneli basit ve kullanışlıdır.
